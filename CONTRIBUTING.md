# Contributing

1. Use Node.js 20 or newer.
2. Run `npm install`.
3. Run `npm test` before opening a pull request.
4. Run `npm pack --dry-run` before release work to confirm the published file set.

Cookiedip intentionally excludes private Justicar orchestration code. Keep queueing, VPN routing, customer data, and deployment-specific logic out of this repository.
